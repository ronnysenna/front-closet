Importar Produtos

Upload de Arquivo JSON/JS
Selecione o arquivo products.js ou products.json da sua aplicação.
Instruções:
  1. O arquivo deve conter um array de produtos em formato JSON.
  2. Se for um arquivo JS, deve estar no formato: export const products = [...]
  3. Cada produto deve ter pelo menos name e price.
  4. Outros campos serão preenchidos com valores padrão se não especificados.
